# Privacy Policy for HaLoo Screenshot Tool

Effective Date: April 29, 2026

HaLoo Screenshot Tool is a Chrome extension designed to help users save order images, original product images, and page screenshots from webpages they actively operate.

This Privacy Policy explains how the extension handles user data.

## Single Purpose

The sole purpose of this extension is to help users save webpage order images, original product images, and page screenshots to their local device. The extension is not used for advertising, user tracking, data resale, or unrelated data collection.

## Data Collection

The extension may process webpage content when the user actively starts the extension workflow. This may include:

- Webpage image elements
- Image URLs
- Order-related page elements
- Page content required for screenshot processing
- User settings saved locally, such as page size, operation delay, save original image option, and page screenshot option

The extension does not collect personal identity information, financial information, health information, authentication information, or other sensitive personal information.

## How Data Is Used

The data processed by the extension is used only to provide the core functionality of the extension, including:

- Detecting images on the active webpage
- Saving original product images
- Creating page screenshots
- Preserving user settings between sessions
- Sending user-initiated commands such as start, stop, inject panel, or clear cache

## Local Storage

The extension may store user preferences locally in the browser, such as operation delay, page size, save original image option, page screenshot option, and panel settings.

This locally stored data is used only to improve the user experience and keep the user’s settings between sessions.

## Downloads

The extension may save images or screenshots to the user’s local device when the user actively starts the workflow. The extension does not download unrelated files without user action.

## Data Sharing

The extension does not sell, rent, transfer, or share user data with third parties.

The extension does not use user data for advertising, tracking, profiling, or analytics.

## Remote Code

The extension does not load or execute remotely hosted JavaScript, WebAssembly, or other executable code. All executable code is packaged with the extension itself.

The extension may read image URLs from webpages or communicate with a local service on the user’s own device, but it does not execute remote hosted code.

## Security

The extension processes data only as needed for its core functionality. Any saved images or screenshots are stored on the user’s local device and are managed by the user.

## User Control

Users can stop the extension workflow at any time from the extension panel. Users can also clear locally stored extension settings through the extension interface or by removing the extension from Chrome.
